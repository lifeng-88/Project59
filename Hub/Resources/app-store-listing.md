### 描述（Description）

```
Hub · Lumina Focus 是一款为「清晰、宁静、有意图」而设计的效率应用。当待办清单变得杂乱，我们帮你把一天重新整理成可执行、可专注、可回顾的秩序。

【今日 · 从清单到行动】
• 一眼看清今天还剩多少任务
• 快速添加：日期、优先级、分类与提醒一次设定
• 搜索、完成、删除——常用操作触手可及
• 侧栏快捷入口：新建任务、开始专注

【日历 · 按天规划】
• 月历视图浏览任意日期的任务安排
• 点击日期即可查看详情或快捷添加
• 空档日一键「添加任务」或「开始专注」

【专注模式 · 番茄钟工作流】
• 可自定义专注、短休、长休时长
• 深度专注模式与环境音（如雨声）营造沉浸感
• 阶段结束本地通知，后台也能按时提醒
• 会话恢复、连续专注天数与完成番茄数统计

【分析 · 看见真实进步】
• 专注时长、连续天数与目标进度可视化
• 下拉刷新，数据随你的使用动态更新
• 激励你保持节奏，而不是制造焦虑

【设置 · 贴合你的习惯】
• 浅色 / 深色 / 跟随系统
• 中 / 英文界面切换
• 专注目标（小时）与番茄钟参数可调
• 任务数据 JSON / CSV 导出与导入
• 可选 iCloud 备份与恢复（需登录 Apple ID）

【设计 · Lumina 宁静蓝】
极简留白、柔和圆角与宁静蓝主色，减少视觉噪音，让任务本身成为焦点。无第三方广告 SDK，不打扰你的专注流。

无论你是职场人、学生还是自由职业者，Hub 都希望成为你设备里那块「数字呼吸空间」——少一分杂乱，多一分从容。

联系我们：privacy@lumina.io
```

### Description

```
Hub · Lumina Focus is a productivity app built for clarity, calm, and intention. When your to-do list feels noisy, we help you turn the day into a manageable sequence of focused actions.

TODAY — FROM LIST TO ACTION
• See how many tasks remain for today at a glance
• Quick add with date, priority, category, and reminders
• Search, complete, and delete with fluid gestures
• Side menu shortcuts for new tasks and focus sessions

CALENDAR — PLAN BY DAY
• Browse tasks on any date in a monthly view
• Tap a day to review or add tasks instantly
• Empty days offer quick “Add task” or “Start focus” actions

FOCUS MODE — POMODORO WORKFLOW
• Customize work, short break, and long break durations
• Deep focus mode and ambient sounds (e.g. rain) for immersion
• Local notifications when a phase ends—even in the background
• Session resume, focus streaks, and completed Pomodoro counts

INSIGHTS — SEE REAL PROGRESS
• Visualize focus time, streaks, and goal progress
• Pull to refresh; stats update as you use the app
• Encouragement without anxiety

SETTINGS — YOUR PREFERENCES
• Light, dark, or system appearance
• Chinese and English UI
• Adjustable focus goals and Pomodoro timers
• Export and import tasks as JSON or CSV
• Optional iCloud backup and restore (Apple ID required)

DESIGN — SERENE BLUE MINIMALISM
Generous whitespace, soft corners, and a restrained Serene Blue palette keep visual noise low so your tasks stay center stage. No third-party ad SDKs—your focus flow stays uninterrupted.

Whether you’re a professional, student, or freelancer, Hub aims to be your digital breathing room: less clutter, more calm.

Contact: privacy@lumina.io
```

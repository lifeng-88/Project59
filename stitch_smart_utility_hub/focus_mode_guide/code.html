什么是番茄工作法？

番茄工作法（Pomodoro Technique）是一种简单易行的延时管理方法。通过将工作时间切分为25分钟的“番茄钟”和5分钟的休息，帮助您保持高强度的专注，同时避免过度疲劳。

番茄钟设置

默认设定为25分钟专注与5分钟休息。您可以根据自己的专注节奏，在设置中自定义时长。建议每完成4个番茄钟进行一次15-30分钟的长休息。

深度专注

进入深度专注模式后，系统将屏蔽所有非紧急通知。这种“零干扰”环境能帮助大脑更快进入“心流”状态，处理逻辑复杂或需要创造力的任务。
